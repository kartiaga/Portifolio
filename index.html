Habilidades

HTML


O HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS


O CSS tem a tarefa de separar o conteúdo do site de sua apresentação visual, alterando elementos como cor do texto, fonte e espaçamento entre blocos, assim como todo o aspecto estético de uma página.

JavaScript


JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.

Figma


O Figma é uma ferramenta grátis muito utilizada pelos designers para a criação de interfaces.

Canva


Canva é uma plataforma de design gráfico que permite aos usuários criar gráficos de mídia social, apresentações, infográficos, pôsteres e outros conteúdos visuais.

Photoshop


O Photoshop é um softwarede imagens que permite que o usuário crie e edite imagens em 2D e 3D

Illustrator


Adobe Illustrator é um editor de imagens vetoriais desenvolvido e comercializado pela Adobe Systems.

After Effects


Adobe After Effects é um programa de criação de gráficos com movimento e efeitos visuais da empresa Adobe Systems. É extensamente usado em pós-produção de vídeo, filmes e DVDs

Premiere


O Premiere Pro é o melhor software de edição de vídeo para criar um conteúdo incrível. É empregado para a edição de vídeos profissionais.

Audition


Adobe Audition é um DAW da Adobe Systems que serve para tratamento de áudio com multi-pistas, ambiente "não-destrutivo" de edição e mistura, assim como edição waveform "destrutiva".
